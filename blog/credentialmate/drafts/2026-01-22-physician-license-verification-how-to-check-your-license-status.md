## How to Verify Your Own License

### State Medical Board Portals

Every state offers online license verification:

| State | Verification URL |
|-------|-----------------|
| California | mbc.ca.gov/breeze |
| Texas | tmb.state.tx.us |
| Florida | flboardofmedicine.gov |
| New York | op.nysed.gov |
| Pennsylvania | pals.pa.gov |
| Ohio | elicense.ohio.gov |

### Information Displayed

| Element | Typically Shown |
|---------|-----------------|
| License number | Yes |
| Status | Active/Expired/Suspended |
| Expiration date | Yes |
| Issue date | Yes |
| Name | Yes |
| Address | Often business only |
| Disciplinary actions | If any |
| Specialty | Sometimes |

| Public | No (authorized users only) |

## What to Do If Information Is Wrong

### Incorrect Information

| Issue | Action |
|-------|--------|
| Wrong name | Contact board with documentation |
| Wrong address | Update through portal |
| Wrong expiration | Verify with board |
| Wrong discipline | Dispute through board |

### Correction Process

1. Identify specific error
2. Gather documentation
3. Contact state board
4. Submit correction request
5. Verify correction made

## Self-Query Best Practices

### Recommended Frequency

| Situation | Check Frequency |
|-----------|-----------------|
| After renewal | Immediately |
| Before credentialing | 30 days prior |
| Annually | Routine check |
| After address change | Verify updated |

### What to Verify

- [ ] Status shows "Active"
- [ ] Expiration date correct
- [ ] Name spelled correctly
- [ ] No unexpected discipline
- [ ] Address current

## Key Takeaways

- **Every state offers free license verification** online
- **DocInfo.org** provides national lookup
- **Check your own license** regularly
- **Verify after renewal** to confirm processed
- **Correct errors promptly** through state board

## Conclusion

Regular license verification protects your career by ensuring your public record is accurate and your status is current. Most verification takes only minutes through state board websites. Make it a habit to verify after every renewal and before any credentialing process.
