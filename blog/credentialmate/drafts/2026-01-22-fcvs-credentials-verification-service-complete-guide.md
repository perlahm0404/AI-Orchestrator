# FCVS Credentials Verification Service: Complete Guide for Physicians

## Introduction

The Federation Credentials Verification Service (FCVS) is a centralized repository for physician credentials that simplifies licensing and credentialing across multiple states. By creating a permanent FCVS profile, you avoid redundant verification requests for medical school, residency, examination history, and other credentials.

## What Is FCVS?

### Overview

| Feature | Details |
|---------|---------|
| Operated by | FSMB |
| Purpose | Primary source credential verification |
| Website | fcvs.fsmb.org |
| Cost | ~$375-500 initial |

### What's Verified

| Credential | Verified |
|------------|----------|
| Medical education | Yes |
| USMLE/COMLEX | Yes |
| Postgraduate training | Yes |
| License history | Yes |
| Board certification | Yes |
| Disciplinary history | Yes |

## Benefits of FCVS

### For Multi-State Licensing

| Benefit | Details |
|---------|---------|
| One-time verification | Credentials verified once |
| Multiple state use | Share with any requesting board |
| Reduced redundancy | No repeat verifications |

### For Credentialing

| User | Benefit |
|------|---------|
| Hospitals | Quick credential verification |
| Insurance panels | Streamlined enrollment |
| Employers | Verified background |

## How FCVS Works

### Creating Your Profile

1. Register at fcvs.fsmb.org
2. Complete application
3. Pay initial fee (~$375-500)
4. Submit required documentation
5. Verification completed
6. Profile available for sharing

### Using Your Profile

1. State board requests FCVS
2. You authorize release
3. Board receives verified credentials
4. Application processed

## Costs

| Fee | Amount |
|-----|--------|
| Initial profile | ~$375-500 |
| Additional state | ~$75-100 |
| Updates | Varies |

## FCVS and IMLC

The IMLC uses FCVS for credential verification, making an FCVS profile beneficial for compact licensing.

## Key Takeaways

- **Create FCVS profile early** in your career
- **One-time investment** saves repeated verifications
- **Essential for multi-state** licensure
- **Speeds credentialing** processes
