# PDMP Registration Requirements for Physicians by State

## Introduction

Prescription Drug Monitoring Programs (PDMPs) are state-run databases that track controlled substance prescriptions. Nearly all states now mandate that prescribers register with and check their state's PDMP before prescribing certain controlled substances. Understanding these requirements is essential for compliance.

## What Is a PDMP?

### Overview

| Feature | Description |
|---------|-------------|
| Purpose | Track controlled substance prescriptions |
| Coverage | Schedule II-V typically |
| Operators | State agencies |
| Users | Prescribers, dispensers, law enforcement |

### Typical Information Tracked

| Data | Tracked |
|------|---------|
| Patient name | Yes |
| Controlled substance | Yes |
| Quantity | Yes |
| Prescriber | Yes |
| Pharmacy | Yes |
| Date | Yes |

## State Requirements

### Registration Mandates

| Requirement | States |
|-------------|--------|
| Mandatory registration | Most states |
| Mandatory check before prescribing | Increasing |
| Specific timeframes | 24-72 hours before prescribing |

### Check Requirements

| State Type | When to Check |
|------------|---------------|
| Mandatory check states | Before every Schedule II script |
| Some states | Schedule II-IV |
| Others | First-time prescriptions |

## Major State PDMPs

| State | PDMP Name | Website |
|-------|-----------|---------|
| California | CURES | oag.ca.gov/cures |
| Texas | Texas PMP | txpmp.org |
| Florida | E-FORCSE | floridahealth.gov |
| New York | I-STOP | health.ny.gov |
| Ohio | OARRS | pharmacy.ohio.gov |

## Compliance Requirements

### Registration Steps

1. Register with state PDMP
2. Complete required training
3. Check before prescribing (if mandatory)
4. Document compliance

### Penalties for Non-Compliance

| Violation | Consequence |
|-----------|-------------|
| Failure to register | Board action |
| Failure to check | Fines, discipline |
| Pattern violations | License action |

## Key Takeaways

- **Register with your state's PDMP** immediately
- **Know check requirements** for your state
- **Document PDMP checks** when required
- **Stay current** on evolving requirements
