# Physician License Application Timeline: What to Expect

## Introduction

Understanding medical license application timelines is crucial for career planning. Whether you're a new graduate, relocating, or adding states for locum work, knowing how long the process takes helps you plan accordingly.

## Typical Processing Times

### New License Application

| Method | Timeline |
|--------|----------|
| Traditional (single state) | 8-16 weeks |
| IMLC (multiple states) | 4-10 weeks |
| Expedited (if available) | 4-8 weeks |

### Factors Affecting Timeline

| Factor | Impact |
|--------|--------|
| Complete application | Faster |
| Incomplete | Significant delays |
| Disclosures | Add 2-8 weeks |
| Background check | 2-4 weeks |
| Board review required | 4-8 additional weeks |

## Application Stages

### Stage 1: Preparation

| Task | Time |
|------|------|
| Gather documents | 1-2 weeks |
| Complete forms | 1-2 days |
| Request verifications | 2-4 weeks |

### Stage 2: Submission

| Task | Time |
|------|------|
| Submit application | 1 day |
| Payment processing | Same day |
| Initial review | 1-2 weeks |

### Stage 3: Verification

| Task | Time |
|------|------|
| Education verification | 2-4 weeks |
| Training verification | 2-4 weeks |
| Exam verification | 1-2 weeks |
| Background check | 2-4 weeks |

### Stage 4: Board Review

| Task | Time |
|------|------|
| Staff review | 1-2 weeks |
| Committee review (if needed) | 2-4 weeks |
| License issuance | 1-5 days |

## State Comparisons

### Faster States

| State | Typical Time |
|-------|-------------|
| States with online systems | 6-10 weeks |
| IMLC states (through compact) | 4-10 weeks |

### Slower States

| State | Typical Time |
|-------|-------------|
| States requiring board review | 12-16 weeks |
| States with paper processes | 12-20 weeks |

## Tips to Speed Processing

| Strategy | Benefit |
|----------|---------|
| Complete application | No delays for follow-up |
| Request verifications early | Parallel processing |
| Use FCVS | Pre-verified credentials |
| Use IMLC | Expedited multi-state |
| Follow up proactively | Identify issues early |

## Key Takeaways

- **Plan 8-16 weeks** for traditional applications
- **IMLC is faster** at 4-10 weeks
- **Complete applications** process fastest
- **Disclosures add time** - plan accordingly
- **Follow up regularly** to track progress
